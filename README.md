# 📊 Excel Project: SaaS Business Performance Dashboard – Document360

This project focuses on analyzing the business performance of **Document360**, a SaaS platform. It uses **Microsoft Excel** for visual dashboards and **a detailed written report** for performance interpretation and strategic recommendations.

---

## 📁 Files Included

- `SaaS_Metrics_Dashboard -13.01.xlsx`  
  ➤ An Excel dashboard containing key SaaS metrics and interactive visualizations.

- `Document360 SaaS Business Performance Analysis & Recommendations.pdf`  
  ➤ A PDF report outlining business KPIs, trends, insights, and recommendations.

---

## 📊 Dashboard Metrics

The Excel dashboard includes analysis on:

- **Monthly Recurring Revenue (MRR)**
- **Churn Rate & Retention**
- **Net Revenue Retention (NRR)**
- **Customer Lifetime Value (LTV)**
- **Customer Acquisition Cost (CAC)**
- **LTV:CAC Ratio**
- **Rule of 40 Score**

---

## 📈 Key Features

- Interactive charts, pivot tables, and dynamic KPIs
- Conditional formatting to highlight critical metrics
- Clean and structured layout for stakeholders and leadership teams

---

## 📄 Report Highlights

The PDF report provides:

- Metric definitions and performance breakdowns
- Strategic recommendations based on data
- Visual evidence supporting business trends
- Recommendations to reduce churn and optimize CAC

---

## 🎯 Outcome

This project showcases a complete SaaS business performance overview combining **data analysis**, **dashboard design**, and **strategic thinking** using only Excel and business logic.

---

## 📌 Tools Used

- Microsoft Excel (PivotTables, Charts, Conditional Formatting)
- Business KPIs for SaaS (MRR, NRR, LTV, CAC)
- Microsoft Word (for documentation)
